# Kurye – Paket Teslim Yönetim Sistemi

Bu proje, **BIL303 Veri Tabanı Yönetimi** dersi final projesi kapsamında geliştirilmiştir.

## Proje Amacı
Kurye firmalarının paket oluşturma, takip, kurye atama ve teslim süreçlerini
SQL Server üzerinde yönetebilmesini sağlayan bir veritabanı sistemidir.

## Kullanılan Teknolojiler
- Microsoft SQL Server 2017+
- SQL Server Management Studio (SSMS)

## İçerik
- DDL: Tablolar, ilişkiler, kısıtlar
- DML: Örnek ve demo veriler
- VIEW: Paket özet ve konum görünümleri
- STORED PROCEDURE:
  - Paket oluşturma
  - Rota ilerletme
  - Otomatik kurye atama
  - Demo veri üretimi

## Kurulum
1. `sql/KuryeTeslimDB.sql` dosyasını SSMS üzerinden çalıştırın.
2. Script idempotenttir, tekrar çalıştırılabilir.
3. Menü bölümünden @AnaSecim değeri değiştirilerek test edilebilir.
